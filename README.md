# Oil-Palm-Ripeness-Detection-App
Developed a mobile-based computer vision system for oil palm fruit ripeness detection using EfficientDet-Lite and MobileNetV2
Implemented real-time detection and image analysis to classify fruits into ripe, unripe, and overripe categories
Converted and optimized deep learning models to TensorFlow Lite (TFLite) for offline mobile deployment
Built a Flutter-based Android application with smooth UI and real-time camera integration
